Projeto: Aprendendo python para dados
Este projeto tem como objetivo o aprendizado de python para a área de dados. O projeto envolve desde a criação de uma API para baixar dados públicos sobre bancos nacionais, criação de alertas, criação de DataFrames, manipulações e correções dos dados, até o produto final para a entrega do curso da Coderhouse.

🚀 Começando:
Esse projeto foi desenvolvido para ser rodado diretamente no Google Colab. Logo, se o usuário decidir utilizar este código em sua máquina local, modificações de compatibilidade
devem ser realizadas para seu funcionamento.

📋 Pré-requisitos:
O único pré-requisito é a criação de uma conta no Google Colab e possuir acesso à internet.
Link para acessar o Google Colab, acesse: https://colab.google

🔧 Instalação:
Não é necessário nenhum tipo de instalação. Apenas executar o código nas opções Getting started> +Code.

⚙️ Executando os testes:
Como teste, digite o código:
print ("hello, world"),para sua primeira conexão com o Google Colab. Após conectado, seu Google Colab já está pronto para executar o código.
Como resultado, o Colab deve gerar uma resposta com: hello, world
